Note 2: If you get any CD Key errors in Gore Special Edition (either CD Key in use or other CD Key related errors) all you have to do is on the Main Menu, click System Options, then click Enter a new Key. Enter any random characters you want and it will work (any gibberish will do, for example goregorewhatever)!

Note 3: Windows might give you a warning when you try to run the Gore Special Edition or Gore Ultimate Soldier installers. Press More Info, then Run Anyway as it is a false alarm.
